# HRM ESOP Demo Project

This project is a Human Resource Management System (HRM) demo for Employee Stock Option Plan (ESOP).

## Project Structure

- **Models**: Contains all the model classes for the project.
- **Controllers**: Contains all the controllers to handle requests.
- **Services**: Contains service classes for business logic.
- **Views**: Contains Razor views for the application.
- **wwwroot**: Contains static files like CSS and JavaScript.

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/khiemnguyen-sys/hrm-esop-demo.git
   ```
2. Navigate to the project directory:
   ```bash
   cd hrm-esop-demo
   ```
3. Restore NuGet packages:
   ```bash
   dotnet restore
   ```
4. Run the application:
   ```bash
   dotnet run
   ```
